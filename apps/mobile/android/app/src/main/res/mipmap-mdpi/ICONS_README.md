# Launcher Icons

Drop your launcher icon PNGs here, named `ic_launcher.png` and `ic_launcher_round.png`.

Required densities (one folder each):
- `mipmap-mdpi/`     — 48x48
- `mipmap-hdpi/`     — 72x72
- `mipmap-xhdpi/`    — 96x96
- `mipmap-xxhdpi/`   — 144x144
- `mipmap-xxxhdpi/`  — 192x192

Generate from a single 1024x1024 source using https://romannurik.github.io/AndroidAssetStudio/icons-launcher.html
or `npx react-native-asset` after dropping the master image in `src/assets/icon.png`.
